# TeleFolders 功能差异报告

**项目**: crazypeace/TeleFolders
**对比基线**: Noradrenalin-team/TeleFolders
**报告日期**: 2026-07-04

---

## 一、新增功能

### 1. CSV 导入 / 导出
- **导出**: 将当前账号的所有聊天与文件夹关系导出为 CSV 文件，采用「宽格式」（每行一个聊天，每列一个文件夹），支持 UTF-8 BOM（Excel 兼容）
- **导入**: 从 CSV 文件恢复文件夹配置，支持增量更新（仅修改变更部分，跳过不变项）、严格的位置匹配、数量与顺序校验
- **状态缩写**: CSV 中支持 `I`/`P`/`X`/`N` 缩写（分别对应 include/pinned/exclude/empty），大小写不敏感

### 2. 创建文件夹
- 通过界面 `+` 按钮弹出对话框，输入名称即可创建新的 Telegram 文件夹
- 新文件夹默认包含所有联系人

### 3. 归档状态切换
- 点击聊天的归档按钮，可将其移入或移出归档
- 操作完成后自动查询服务器确认实际生效状态，并在 UI 同步更新图标

### 4. 启动脚本（start.sh）
- 提供一键启动脚本，内置 flock 单实例锁，防止多个进程竞争端口
- 进程崩溃后自动重启
- API 凭证通过环境变量传入，不再硬编码

### 5. 纯 HTTP 运行模式
- 后端以纯 HTTP 服务器模式运行（`mode=None`），用户可通过任意浏览器访问 `http://<IP>:8000/main.html`
- 不再依赖 Eel 内置浏览器

### 6. 命令行语言参数
- 启动时通过 `--lang en` 或 `--lang ru` 指定界面语言
- 前端自动从服务器读取当前语言设置并同步

### 7. 调试与日志
- 后端输出运行状态日志（聊天数量、文件夹数量、操作结果等）
- 便于排查问题

### 8. 登录流程优化
- 连接失败或会话被撤销时，显示登录表单而非错误弹窗
- `get_user()` 增加了空值保护，避免客户端未初始化时崩溃

### 9. Eel 稳定性增强
- 兼容 Telethon 不同版本的返回格式（`result.filters` 兜底）
- 文件夹标题支持 `TextWithEntities` 格式提取

---

## 二、移除 / 未包含的功能

### 1. 联系人分类显示
- 原后端调用 `get_contacts()` 区分「联系人」与「非联系人」用户
- 因 Telethon 1.44.0 移除了该方法，且前端从未消费该字段，现已完全移除

### 2. 界面内语言切换
- 原 UI 提供语言切换按钮
- 现改为仅通过启动参数 `--lang` 设定，简化 i18n 逻辑

---

## 三、架构与依赖变更

| 项目 | Noradrenalin | crazypeace |
|------|-------------|-----------|
| Eel 来源 | Git fork (`philipdp123/Eel`) | PyPI `eel>=0.18.0` |
| Telethon 版本 | 1.34.0 | 1.44.0 |
| 运行模式 | 内置浏览器 (`mode="default"`) | 纯 HTTP (`mode=None`) |
| 启动入口 | `python -m telefolders` | `start.sh` 或 `python -m telefolders --lang en` |
| 语言切换 | 界面按钮 | 启动参数 |
| 注释语言 | 俄语 | 英语 |
| i18n 引擎 | 无独立模块 | `i18n.js` + JSON 翻译文件 |
| 安全 | API 硬编码 | 环境变量传入 |

---

## 四、UI 变更

1. **CSV 菜单**: Header 用户菜单新增「Export CSV」和「Import CSV」选项
2. **文件夹创建**: 点击 `+` 列弹出输入框，确认后创建
3. **国际化文本**: 所有界面元素通过 `data-i18n` 属性标记，翻译文本存储在 `/locales/en.json` 和 `/locales/ru.json`
4. **标题显示**: 正确处理 Telegram 文件夹标题中的富文本格式

---

## 五、安全性

- API ID / Hash 不再硬编码在代码中，通过环境变量 `${TELEFOLDERS_API_ID}` / `${TELEFOLDERS_API_HASH}` 传入
- 会话文件已加入 `.gitignore`，不会提交到仓库

---

*本报告仅描述 crazypeace/TeleFolders 的功能变动，不描述 Noradrenalin-team/TeleFolders 的原有功能。*
